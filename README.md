1. DATA SOURCE 
This code is designed for the "Intracardiac Atrial Fibrillation Database" (iafdb).
Source: https://physionet.org/content/iafdb/1.0.0/

2. SETUP 
* Download files from PhysioNet.
* Place them in the same folder as the script.

3. HOW TO RUN 
1. Open the script and set the 'nome' variable (e.g., "iaf5_tva").
2. Run the code.

4. RESULTS 
Pre-analyzed results are available in these folders:
* [Regularity Index]: Screenshots of ρ evolution and drug injections.
* [Waveform]: Screenshots of LAWs patterns.
